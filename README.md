# md_project — QLN 2.0: AI-Native Instruction Language

> Rust + MD + SQL + Llama Semantic = 차세대 AI 에이전트 지시 언어

MD의 가독성, Rust의 타입 안전성, SQL의 구조화 쿼리, Llama의 시맨틱 이해를 결합한 신규 언어 개발 프로젝트.

## 구조

```
md_project/
├── docs/          # 프로젝트 문서 (기획, 리서치, 아키텍처)
├── compiler/      # Rust 기반 컴파일러/파서 (추후)
├── runtime/       # 런타임 엔진 (추후)
└── examples/      # 예제 .n2 파일 (추후)
```

## 핵심 목표

1. **MD의 한계 극복**: 강제력 있는 규칙 시스템
2. **QLN 1.0 → 2.0 진화**: 세분화된 레이어 아키텍처
3. **결정론적 출력**: 같은 입력 → 같은 결과 보장
